# 银河莉萝放置 - 游戏概述

## 游戏类型

银河莉萝放置是一款多人在线放置增量游戏（Idle Game），灵感来源于 Melvor Idle 和 Your Chronicle。玩家通过设置行动队列，角色会自动执行采集、制造、战斗等活动，即使离线也会持续积累进度。

## 核心玩法循环

```
设置行动队列 → 离线/在线挂机 → 同步结算 → 获得资源/经验 → 制造/升级装备 → 解锁新内容 → 设置更高级队列
```

## 行动类型

游戏中的所有行动分为三大类：

### 循环行动（Loop）
- 可以反复执行的行动，如砍树、挖矿、制造物品
- 每次执行消耗固定时间（`loop_time`），消耗材料并产出奖励
- 放入队列后会持续循环执行，直到材料耗尽或被玩家更换

### 即时行动（Instant）
- 一次性执行的行动，立即完成
- 消耗材料并产出奖励，不消耗时间

### 升级行动（Upgrade）
- 一次性解锁的里程碑，解锁后永久生效
- 包括剧情对话、建筑建造等
- 已解锁的升级行动不可重复执行

## 技能体系

| 技能 ID | 名称 | 说明 |
|---------|------|------|
| felling | 伐木 | 砍伐各类树木获取木材 |
| mining | 采矿 | 挖掘泥土、石头、矿石 |
| crafting | 制造 | 制作木板、工具、装备 |
| forging | 锻造 | 熔炼、冶金、凝结材料 |
| enhancing | 强化 | 打磨和强化装备 |

每个技能拥有独立的等级和经验值。执行关联技能的行动会获得该技能的经验值，达到阈值后自动升级。更高的技能等级可以解锁更高级的行动。

## 资源类型

### 物品（Items）
可堆叠的背包物品，包括：
- **资源类**（resources）：橡木原木、橡木木板、木棍等
- **矿石类**（ores）：石头、铁矿石、黑石、铁锭等
- **燃料类**（fuel）：煤炭（附带热量值）
- **动物材料**（animal_materials）：毛皮、皮革、生肉等
- **装备类**（equipment）：武器、防具
- **工具类**（tool）：斧、镐、工作台
- **重要物品**（important）：空间、绿宝石

### 流体（Fluids）
特殊的不可堆叠资源，用于高级锻造配方：
- `stone_fluid`：石流体
- `iron_fluid`：铁流体
- `heat`：热量

## 地图与区域

当前所有行动发生在 `village`（村庄）区域。未来将开放更多地图区域。

## 游戏进程概览

1. **新手引导**：完成与神秘人/"妈妈"的对话（5段），了解游戏机制
2. **基础采集**：砍伐橡木、挖掘泥土，积累基础资源
3. **制造入门**：制作木板 → 木棍 → 木制工具/武器
4. **采矿升级**：挖掘石头、寻找铁矿
5. **家园建设**：消耗空间和材料建造农田、熔炉等设施
6. **锻造体系**：制作木炭、熔炼黑石、凝结铁锭
7. **装备进阶**：石制工具/武器、皮革防具
8. **战斗系统**：屠宰场战斗，获取动物材料
9. **强化系统**：装备附魔、强化、赋能（剧情铺垫中）
