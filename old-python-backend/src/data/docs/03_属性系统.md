# 属性系统

## 概述

属性系统是角色成长的核心。玩家通过装备武器、防具和工具，获得各种属性加成。这些属性影响战斗表现、采集效率、经验获取和奖励产出。

## 等级

等级是衡量

### 等级带来的属性提升

每 1 级生产等级, 给予 0.3% 的乘算行动产出: 例如 101 级将给予 1.003^100 = 1.35 的等级效率加成. (赋能等级除外; 在 600 级左右和市面标准的每一级 1% 加成重合)

赋能等级直接影响装备附魔上限和成功率; 每个强化都带有 推荐等级 词条, 当赋能等级等于 推荐等级 时, 维持基础成功率, 高于时获得平方关系的成功率加成, 低于时获得指数关系的成功率降低效果. 具体而言, 计算公式如下:

```
等级大于推荐等级时:
成功率 = 基础成功率 * ( 1 + ( 赋能等级 - 推荐等级 ) / 35 )^2 * ( 1 + 装备提供的最终成功率倍率加成 )
等级小于推荐等级时:
成功率 = 基础成功率 * 0.99 ^ ( 推荐等级 - 赋能等级) * ( 1 + 装备提供的最终成功率倍率加成 )
```

- 批量强化的逻辑是, 比如有数个 +0 的武器强化到 +10, 则会将所有物品都强化到 +1, 再强化到 +2, 而非一个一个强化到 +10
- 当成功率大于 100% 时, 如果此时正在批量强化, 则有 (成功率 - 100%) 的概率直接为下一个物体强化完毕 (若成功仍然消耗材料, 未成功则不扣除), 若还是大于 100% 重复以上步骤直到本次成功率小于等于 100% 或者当前批次强化完毕. 

每 1 级力量等级, 给予 1 点物理基础伤害加算加成 和 0.5% 的物理基础伤害乘算倍率
每 1 级远程等级, 给予 0.5 点精确值和闪避值加算加成
每 1 级坚韧等级, 给予 5 点基础血量上限
每 1 级耐力等级, 给予 0.02 点耐力自然回复 和 1 点耐力上限
每 1 级智力等级, 给予 0.02 点魔力自然回复 和 1 点魔力上限
每 1 级防御等级, 给予 1 点防御值和格挡值加成
每 1 级魔法等级, 给予 1 点奥术基础伤害加算加成 和 0.5% 的奥术基础伤害乘算倍率

### 等级曲线

单级升级所需经验值计算公式为:

单级所需经验 = 800 * 等级 * 1.01 ^ 等级 / ( 1 + 9 / (等级 ^ 0.5) )

## 属性来源

### 装备（Equipment）
装备提供**固定值加成**（flat modifier）。例如：
- 木剑：物理伤害 +10，命中 +10，攻击间隔 2.0
- 皮革胸甲：防御 +8
- 克苏鲁木雕：经验获取倍率 +0.1

### 工具（Tool）
工具提供**百分比加成**（percent_add modifier）。例如：
- 木斧：伐木生产倍率 +10%
- 石镐：采矿生产倍率 +20%
- 石工作台：制造生产倍率 +20%

## 修饰器类型

属性系统使用三层修饰器，按以下顺序叠算：

| 修饰器类型 | 说明 | 叠加方式 |
|-----------|------|---------|
| `flat` | 固定值 | 直接相加 |
| `percent_add` | 百分比加算 | 先求和，再一次性乘算 |
| `percent_mult` | 百分比乘算 | 逐个连乘 |

## 最终属性公式

```
最终值 = (基础值 + 等级效果 + Σ 装备加算 ) × Π(1 + 乘算_i)
```

注: 所有的乘算效果都是以 1 = 100% 计算, 即不带有百分比

部分装备的 `equipment_position_requirements` 中 `position` 为 `"nothing"`，表示该装备不消耗装备槽位, 但是只能装备一个（如克苏鲁木雕、棉制白丝）。

- 你可以看成全局 buff, 后续也可能会调整.

## 性能特性

- 属性集在每次结算开始时构建一次（O(N)，N 为修饰器数量）
- 后续所有属性查询均为 O(1)
- 实测：10,000 次构建 + 100,000 次查询 < 0.15 秒

## 属性计算

计算公式(生产):

工具的属性面板由 impl.ts 的 tool_basic_data 接口定义.

单个技能产出加成 = ( 1 + 求和 (身上所有工具的对应 production_multiplier 属性) ) * 等级带来的效率 (有些装备会提供等级加成, 因此计算的等级 = 用户等级 + 身上所有装备的对应 level_buff 属性 )

单个技能速度加成 = 求积 (1+身上所有工具的对应 speed_multiplier 属性 )

计算公式(战斗):

装备的属性面板由 impl.ts 的 equipment_basic_data 接口定义.

用户生命值上限 (hp)= ( 基础生命值上限 + 5 * 坚韧等级 + 求和 (身上所有装备的 maxhp 属性) ) * 求积(1 + 身上所有装备的 maxhp_multiplier 属性)

用户耐力值上限 (sp)= ( 基础耐力值上限 + 1*耐力等级 + 求和 (身上所有装备的 maxsp 属性) ) * 求积(1 + 身上所有装备的 maxsp_multiplier 属性)

用户魔力值上限 (mp) = ( 基础耐力值上限 + 1*耐力等级 + 求和 (身上所有装备的 maxmp 属性) ) * 求积(1 + 身上所有装备的 maxmp_multiplier 属性)

用户物理攻击力 (physical_power) = ( 基础物理攻击力 + 1*力量等级 + 求和 (身上所有装备的 physical_power 属性)) * 求积(1 + 身上所有装备的 power_multiplier 属性) * (1.005) ^ 力量等级

用户魔法攻击力 (magic_power) = ( 基础攻击力 + 1*力量等级 + 求和 (身上所有装备的 magic_power 属性)) * 求积(1 + 身上所有装备的 power_multiplier 属性) * (1.005) ^ 力量等级

用户攻击速度 (attack_interval) = max(身上所有武器的 attack_interval 属性) / 求积(1+ 身上所有装备的 attack_speed 属性)

用户暴击率 (critical) = min ( (  基础暴击率 + 求和 (身上所有装备的 critical 属性) )* 求积 (1+身上所有装备的 critical_possibility_multiplier 属性) , 1 )

用户暴击倍率 (critical_rate) = 基础暴击倍率 +  求和 (身上所有装备的 critical_rate 属性)

用户格挡值 (block) = ( 基础格挡值 +  1 * 防御等级 + 求和 (身上所有装备的 block 属性) )* 求积 (1+身上所有装备的 block_multiplier 属性)

用户最终格挡概率加成 (block_possibility_multiplier) = ( 1 + 基础加成) * 求积 ( 1 + 身上所有装备的 block_possibility_multiplier 属性)

用户格挡倍率 (block_rate) =( 基础格挡倍率 + 求和 (身上所有装备的 block_rate 属性) ) * 求积 (1+身上所有装备的 block_rate_multiplier 属性)

用户生命回复 (hp_recovery) = ( 基础生命回复 + 求和 (身上所有装备的 hp_recovery 属性) ) * 求积 (1+ 身上所有装备的 overall_recovery_speed 属性)

用户耐力回复 (sp_recovery) = ( 基础耐力回复 + 0.02 * 耐力等级 + 求和 (身上所有装备的 sp_recovery 属性) ) * 求积(1 + 身上所有装备的 overall_recovery_speed 属性)

用户法力回复 (mp_recovery) = ( 基础耐力回复 + 0.02 * 耐力等级 + 求和 (身上所有装备的 sp_recovery 属性) ) * 求积(1 + 身上所有装备的 overall_recovery_speed 属性)

用户精准值 (accuracy) = ( 基础精确值值 +  0.5 * 远程等级 + 求和 (身上所有装备的 accuracy 属性) )* 求积 (1+身上所有装备的 accuracy_multiplier 属性)

用户最终命中概率加成 (accuracy_possibility_multiplier) = ( 1 + 基础加成) * 求积 ( 1 + 身上所有装备的 accuracy_possibility_multiplier 属性)

用户闪避值 (evade) = ( 基础精确值值 +  0.5 * 远程等级 + 求和 (身上所有装备的 evade 属性) )* 求积 (1+身上所有装备的 evade_multiplier 属性)

用户最终闪避概率加成 (evade_possibility_multiplier) = ( 1 + 基础加成) * 求积 ( 1 + 身上所有装备的 evade_possibility_multiplier 属性)

用户奥术抵抗 (magic_instance)= (基础奥数抵抗 + 求和 (身上所有装备的 magic_instance 属性)) * 求积 (1 + 身上所有装备的 magic_instance_multiplier 属性)

用户防御值 (defense) = ( 基础防御值 + 1 * 防御等级 + 求和 (身上所有装备的 defense 属性) ) * 求积 ( 1 + 身上所有装备的 defense_multiplier 属性)

用户仇恨值 (hatred) = ( 基础仇恨值 + 求和 (身上所有装备的 hatred 属性) ) * 求积 ( 1 + 身上所有装备的 hatred_multiplier 属性)

用户最终伤害加成 (final_damage_multiplier) = 求积 ( 1 + 身上所有装备的 final_damage_multiplier 属性)

用户最终伤害减少 (final_damage_reduce) = 求积 ( 1 + 身上所有装备的 final_damage_reduce 属性)
