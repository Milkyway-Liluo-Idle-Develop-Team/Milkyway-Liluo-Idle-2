|------|------|---------|

### 装备操作

#### 穿戴装备
1. 先结算当前进度
2. 检查物品存在于背包中
3. 检查物品可以装备到指定槽位
4. 从背包中扣除 1 个物品
5. 如果该槽位已有装备，将旧装备退回背包
6. 装备新物品到槽位
7. 重新计算属性

#### 卸下装备
1. 先结算当前进度
2. 检查指定槽位有装备
3. 将装备物品退回背包
4. 清空槽位
5. 重新计算属性

## 工具系统

### 工具与装备的区别

| 特性 | 装备 | 工具 |
|------|------|------|
| 属性加成类型 | 固定值（flat） | 百分比（percent_add） |
| 影响范围 | 战斗属性 | 生产效率 |
| 槽位 | 5个身体部位 | 3个技能槽位 |
