# 行动数据参考

## 行动数据格式

每个行动（event）包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识符 |
| type | string | 类型：`loop` / `instant` / `upgrade` |
| name | string | 显示名称 |
| description | string | 描述文本 |
| need_skill | string | 关联技能 ID，或 `"none"` |
| requirements | array | 前置需求列表（可选） |
| loop_time | number | 循环行动的基础耗时（秒）（仅 loop） |
| combat | boolean | 是否为战斗行动（可选） |
| enemy | object | 敌人数据（仅战斗行动） |
| rewards | array | 奖励列表（可选） |
| experience | number | 基础经验值 |
| map | string | 所属地图区域 |
