# 左侧属性面板

左侧固定面板展示玩家的生产技能等级、升级进度和战斗属性，采用可折叠分组设计。

## 布局

```
┌──────────────┐
│ 生产属性  [收起]
├──────────────┤
│ 砍伐      76  │  ← 圆形等级指示
│ 采矿     118  │
│ 种植       1  │
│ 制造      54  │
│ 锻造       1  │
│ 赋能       1  │
├──────────────┤
│ 升级行动      │
│ 已解锁 1 项   │  ← [UP] 图标
├──────────────┤
│ 战斗属性  [收起]
├──────────────┤
│ 战斗等级 15.71│
│ 力量      34  │
│ 远程      18  │
│ 坚韧      10  │
└──────────────┘
```

## 生产属性

- **可折叠**：点击 `"生产属性"` 标题可收起/展开整个分组。
- **技能列表**：按 `player_skills` 中已解锁的技能展示，每项包含：
  - 技能名称（如"砍伐"、"采矿"、"种植"等）
  - 当前等级数值，置于圆形/环形指示器中
- **等级展示**：数值为整数，圆形背景色根据技能类型区分。
- **点击交互**：点击某项技能可展开该技能的详细经验条和解锁事件预览（可选弹层或展开动画）。

## 升级行动

- 汇总展示玩家通过 upgrade 事件解锁的内容数量。
- 显示 `"已解锁 X 项"`，配合 `UP` 图标。
- 点击可跳转或弹出升级/解锁内容列表。

## 战斗属性

- **可折叠**：同生产属性。
- **战斗等级**：显示综合战斗等级（如 `15.71`），由属性系统计算得出。
- **属性列表**：展示核心战斗属性及其数值，如：
  - 力量
  - 远程
  坚韧
- 数值同样置于圆形指示器中，颜色可与属性方向（positive/negative）关联。

## 实时更新

- 技能等级变化时，对应项数值变化并播放缩放动画。
- 经验获得时，若对应技能面板处于展开状态，经验条平滑增长。
