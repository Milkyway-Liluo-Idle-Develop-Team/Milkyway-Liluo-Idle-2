# 右侧库存面板

右侧固定面板展示玩家拥有的物品和装备，采用分类卡片式布局。

## 标签切换

- **物品**：展示 `player_inventory` 中 `classification != "equipment"` 的物品。
- **装备**：单独展示已穿戴和未穿戴的装备，支持穿戴/卸下操作。

## 分类展示

物品按 `classification` 分组展示，每组有标题：

| 分类 | 说明 |
|------|------|
| 资源 (`resources`) | 泥土、木头、石头等原材料 |
| 矿物 (`ores`) | 铁矿石等 |
| 装备 (`equipment`) | 仅在"装备"标签下展示，或作为独立分组 |
| 动物材料 (`animal_materials`) | 羽毛、毛皮、肉类等 |
| 流体 (`fluid`) | 热能等，数量支持小数展示 |

## 物品卡片

每组内的物品以网格卡片排列：

| 元素 | 说明 |
|------|------|
| 图标 | 物品图标，居中 |
| 名称 | 物品名称，图标下方 |
| 数量 | 蓝色高亮数字，支持 `k`/`M` 缩写（如 `118k`、`1.07k`）|

### 数量缩写规则

| 范围 | 展示 |
|------|------|
| < 1,000 | 完整数字（如 `453`、`827`）|
| 1,000 ~ 999,999 | `k` 缩写，保留1位小数（如 `118k`、`93.9k`）|
| ≥ 1,000,000 | `M` 缩写，保留1位小数 |

## 交互设计

- **悬停**：显示物品完整数量tooltip（不缩写）。
- **点击**：弹出物品详情弹窗（属性、来源、用途等）。
- **装备标签**：装备卡片额外显示 `"[已穿戴]"` 标签，点击可卸下。

## 实时更新

- 事件产出/消耗时，对应物品卡片数量实时变化，播放缩放高亮动画。
- 新获得物品（数量从0变为正数）以渐入动画加入对应分类。
- 数量归零的物品保留显示 `×0` 或渐出移除（根据配置）。
