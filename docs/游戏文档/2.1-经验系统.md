# 经验系统

经验系统负责记录和管理玩家在各技能上的成长进度。

## 技能定义

当前游戏中的技能（`skill`）包括：

| 技能 ID | 名称 | 说明 |
|---------|------|------|
| `crafting` | 制作 | 制作木板、工具、装备等 |
| `enhancing` | 强化 | 装备升级与附魔相关 |
| `felling` | 伐木 | 采集木材类资源 |
| `forging` | 锻造 | 冶炼金属、凝结黑石等 |
| `magic` | 魔法 | 法术相关 |
| `mining` | 采矿 | 采集矿石、泥土、石头等 |
| `strength` | 力量 | 战斗与物理相关 |
| `none` | 通用 | 无特定技能关联的事件 |

每个技能拥有独立的等级（`level`）和经验值（`xp`）。

## 经验获取

经验通过执行事件获得。在 `actions.json` 中，事件的 `rewards` 数组里可以包含 `type` 为 `"experience"` 的奖励：

```json
{
  "type": "experience",
  "value": 20.0,
  "skill_id": "felling"
}
```

- `value`：获得的经验数值（支持小数）。
- `skill_id`：经验归属的目标技能。

事件每次完成循环（`loop` 类型）或执行（`instant` 类型）时，对应的经验奖励会累加到玩家该技能的经验池中。

## 技能升级

当技能经验值达到当前等级上限时，技能等级提升。升级可能带来：

- 解锁更高等级要求的新事件。
- 提升相关事件的效率（如循环时间缩短、产出增加）。
- 满足装备穿戴或工具使用的前置条件。

### 等级曲线

等级曲线通过 `level_curve_csv` 提供，是一个累积经验阈值表：

```
level,xp
1,0
2,100
3,250
4,500
...
```

`LevelCurve` 是一个有序切片，`nextLevel(oldLevel, currentXP)` 从 `oldLevel` 开始向前遍历，找到 `currentXP` 达到的第一个等级阈值。对于小 Tick 增量，均摊时间复杂度接近 O(1)。

### AddXP 流程

```go
func (s *State) AddXP(skillID gameconfig.SkillID, xp float64) {
    slot := s.mustSlot(skillID)
    oldLevel := slot.Level
    slot.XP += xp
    newLevel := s.curve.nextLevel(oldLevel, slot.XP)
    slot.Level = newLevel
    s.dirty[skillID] = true
    s.record(skillID, xp, newLevel)   // 向 recorder 写入 SkillXpRecord

    if newLevel > oldLevel && s.OnLevelUp != nil {
        s.OnLevelUp(skillID, oldLevel, newLevel)   // 触发升级回调
    }
}
```

`OnLevelUp` 回调由 `PlayerSession` 设置，用于在技能升级时更新属性修饰器（技能等级影响属性）。

## 数据存储

玩家技能数据存储在 `player_skills` 表中：

| 字段 | 类型 | 说明 |
|------|------|------|
| `user_id` | INTEGER | 玩家 ID |
| `skill_id` | INTEGER | 技能数字 ID |
| `level` | REAL | 当前等级 |
| `xp` | REAL | 累计经验值 |
| `updated_at` | DATETIME | 最后更新时间 |

主键为 `(user_id, skill_id)`，即每个玩家对每个技能只有一行记录。

### 加载与刷盘

```go
func Load(ctx context.Context, q *dbgen.Queries, userID int64, curve LevelCurve) (*State, error)

func (s *State) Flush(ctx context.Context, q *dbgen.Queries) error {
    for id := range s.dirty {
        slot := s.skills[id]
        q.UpsertSkill(ctx, dbgen.UpsertSkillParams{
            UserID: s.userID, SkillID: int64(id),
            Level: slot.Level, Xp: slot.XP,
        })
    }
    s.dirty = make(map[gameconfig.SkillID]bool)
}
```

## 更新记录

### 技能经验记录

```go
type Bucket struct {
    changes map[gameconfig.SkillID]*skillRec
}

type skillRec struct {
    xpDelta  float64
    newLevel float64
}
```

**产生时机**：事件结算发放经验奖励、其他系统直接给予经验时写入。

**合并规则**：同一 `SkillID` 在同一命名空间内累加 `xpDelta`，`newLevel` 取最终值。

### 序列化示例

```json
{
  "skill_xp": [
    {"skill_id": 4, "xp_delta": 20.0, "new_level": 1.0}
  ]
}
```
