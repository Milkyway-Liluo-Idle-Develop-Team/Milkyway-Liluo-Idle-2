# 概述

本作是一款放置类（Idle）角色扮演游戏，核心玩法围绕**事件队列**展开：玩家选择各种行动（采集、制造、解锁剧情等）放入队列，服务器按时间推进自动结算资源产出与经验积累。

## 核心循环

1. **选择事件**：从当前已解锁的事件列表中挑选目标行动。
2. **投入队列**：将事件放入队列中开始执行。
3. **时间推进**：推进事件和战斗等系统，产出资源和经验。
4. **解锁新内容**：通过 `upgrade` 类型事件解锁更高级的事件、物品或剧情。
5. **强化角色**：升级技能、强化装备、制作更好的工具，提升事件效率。

## 关键设计原则

- **服务端主导结算**：所有游戏进度由服务端计算，客户端仅负责展示和发起操作请求。玩家离线期间的事件产出会在下次同步时一次性结算。
- **实时同步**：客户端通过 WebSocket 维持长连接，关键操作和进度推进通过消息交互完成。
