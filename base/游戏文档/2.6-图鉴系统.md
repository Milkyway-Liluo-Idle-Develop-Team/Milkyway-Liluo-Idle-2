# 图鉴系统

图鉴系统用来记录玩家解锁了的事件、物品等内容。

## 事件图鉴

事件第一次被触发的时候解锁相关图鉴。

## 物品图鉴

第一次获得某种物品的时候解锁相关图鉴。

## 区域图鉴

第一次在相关区域内执行行动的时候解锁。

## 更新记录

图鉴系统在玩家解锁新内容时向当前命名空间写入记录。

### 图鉴解锁记录

```go
type BestiaryUnlockRecord struct {
    Type string    // "event" / "item" / "area"
    ID   string    // 解锁对象的字符串标识（前端展示用）
}
```

**产生时机**：事件首次触发、物品首次获得、区域首次进入时写入。若该内容已在图鉴中，不产生记录。

### 合并规则

同一 `Type` + `ID` 在**同一命名空间**内去重，仅保留一条。

### 序列化示例

```json
{
  "bestiary_unlocks": [
    {"type": "event", "id": "felling_oak_tree"},
    {"type": "item", "id": "oak_logs"}
  ]
}
```
