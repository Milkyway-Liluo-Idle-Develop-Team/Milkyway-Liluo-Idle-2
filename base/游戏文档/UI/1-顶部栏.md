# 顶部栏 (TopBar)

顶部栏固定于页面最上方，贯穿全宽，展示全局状态与快捷操作。

## 布局

```
┌─────────────────────────────────────────────────────────────┐
│ [游戏图标] 银河莉萝放置        砍伐橡木  [=========>    ] 35%
│ Milkyway Liluo Idle            可循环 ∞  [停止]
│                                队列 (1) ▼
│                                1 砍伐橡木 [当前]
│
│                    战斗中…… [村庄 第 1179 波]   [刷新] [主题] [用户] [登出]
└─────────────────────────────────────────────────────────────┘
```

## 元素说明

### 左侧：品牌区

| 元素 | 说明 |
|------|------|
| 游戏图标 | 方形图标，点击可返回默认视图 |
| 中文标题 | "银河莉萝放置"，主标题 |
| 英文副标题 | "Milkyway Liluo Idle"，较小字号 |

### 中间：当前执行状态

| 元素 | 说明 |
|------|------|
| 事件名称 | 当前队列头部正在执行的事件名称，如"砍伐橡木" |
| 进度条 | 水平进度条，显示 `progress / loop_time` 百分比，如 35% |
| 循环标签 | `"可循环 ∞"` 或 `"目标: 10"` 等，表示当前事件的循环状态 |
| 停止按钮 | 点击可暂停/停止当前循环（若游戏支持暂停机制）|

### 队列快捷下拉

- 显示当前队列事件数量，如 `"队列 (1) ▼"`。
- 展开后列出队列中的事件，当前执行事件标记 `"[当前]"`。
- 点击可快速跳转到队列面板。

### 状态文字

- 展示当前游戏状态，如 `"战斗中…… [村庄 第 1179 波]"`。
- 状态变化时（如进入/离开战斗）平滑过渡文字。

### 右侧：功能按钮

| 按钮 | 说明 |
|------|------|
| 刷新 | 手动请求完整状态同步（调试用或异常恢复）|
| 主题 | 切换浅色/深色主题 |
| 用户 | 点击展开用户菜单（设置、帮助等）|
| 登出 | 断开 WebSocket 连接，返回登录页 |

## 实时更新

- 进度条每秒根据服务端推送更新，平滑动画过渡。
- 当前事件完成切换时，事件名称和进度条同步更新。
