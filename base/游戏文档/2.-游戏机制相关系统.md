## 系统间关系

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│   事件系统   │ ──→  │   经验系统   │      │   属性系统   │
│  (Event)    │      │  (Skill)    │      │ (Attribute) │
└──────┬──────┘      └──────┬──────┘      └──────┬──────┘
       │                    │                    │
       └────────────────────┼────────────────────┘
                            ↓
                     ┌─────────────┐
                     │   物品系统   │
                     │  (Item/Inv) │
                     └─────────────┘
```

- **事件系统**是驱动核心：事件执行产生经验和物品。
- **经验系统**记录技能成长，技能等级反过来作为部分事件的执行门槛。
- **物品系统**存储产出资源，装备和工具提供属性加成。
- **属性系统**定义角色和装备的能力数值，影响战斗和事件效率。
- **区域系统**对事件进行分类，新区域通过剧情事件逐步解锁。
